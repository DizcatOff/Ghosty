Good to know

  • Elite only stays the same and is limited to the premium-only list.
  • Client features affect your local game, while Server and TMP features affect online behavior.
  • Warning means there is a known detection risk, and Deprecated means the feature is old and should be treated carefully.
  • Suspension is still included as Front / Rear Suspension in the current list.
