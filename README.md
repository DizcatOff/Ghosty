# Ghosty Website

Official source for the **Ghosty** website.
This site provides information, downloads, and support links for the Ghosty project.

## Overview

Ghosty is a tool designed for use with TruckersMP and single-player environments.
The website serves as the main hub for:

* Feature overviews
* Download links
* Support resources
* Community Information/access

## Project Structure

```
ghosty-website/
├── index.html
├── features.html
├── spoofer.html
├──video.mp4
└── README.md
```

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/ghosty-website.git
cd ghosty-website
```

### 2. Open locally

You can open the site directly in your browser:

* Double-click `index.html`, or
* Use a local server (recommended)

Example with Python:

```bash
python -m http.server 8000
```

Then visit:

```
http://localhost:8000
```

## Deployment

This site is compatible with:

* GitHub Pages
* Netlify
* Vercel
* Any static hosting provider

For GitHub Pages:

1. Push to the `main` branch.
2. Enable Pages in repository settings.
3. Set source to `/root`.
